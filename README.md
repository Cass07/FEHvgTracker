# FEHvgTracker (파엠히 투표대전 트래커)   
## 전체 페이지   
[Click!](https://feh.wiki/voting/vg/)

 * 메인 페이지   
 현재 진행되는 투표대전 현황 OR 제일 마지막으로 종료된 투표대전 결과
 * 5시 초동 데이터    
 제일 마지막 투표대전 데이터의 초동 5시 데이터
 * 전체 데이터 조회    
 저장되어 있는 전체 투표대전 데이터
